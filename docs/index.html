Interstate Traffic Volume Analysis

This project analyzes hourly interstate traffic volume using the Metro Interstate Traffic Volume dataset to uncover repeatable patterns in human mobility. The figures show how demand changes across the day, differs between weekdays and weekends, and responds to weather conditions. The emphasis is interpretabilityturning raw data into visual evidence that supports infrastructure planning and decision-making.
