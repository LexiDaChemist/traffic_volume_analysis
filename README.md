# Energy Consumption Forecasting (Java)

A Java-based time-series forecasting project using household electric power
consumption data. The goal is to build a simple forecasting engine, implement
baseline models, and evaluate prediction accuracy.

**Dataset:** Household Energy Consumption  
**Source:** Kaggle  
**Contributor:** George Okello  
All rights belong to the original dataset creators.
